# StruoCV – Resume Builder

StruoCV est une application web interactive qui permet aux utilisateurs de créer, personnaliser et télécharger facilement un CV professionnel
à travers un formulaire à étapes simple et fluide.

# Fonctionnalités principales

-Formulaire multi-étapes (informations personnelles, compétences, expériences, etc.)
-Ajout dynamique des sections (compétences, langues, formations, expériences, certifications…)
-Validation des champs (email, téléphone, liens, etc.)
-Barre de progression indiquant l’avancement
-Deux modèles de CV au choix avec aperçu en temps réel
-Téléchargement ou impression du CV final au format PDF

# Technologies utilisées

-HTML5
-CSS3 / TailwindCSS
-JavaScript (DOM natif)
-Git / GitHub

# Objectif du projet

Créer une application web moderne et responsive permettant à tout utilisateur de générer un CV clair, structuré et professionnel,
tout en pratiquant la manipulation du DOM, la gestion d’un formulaire dynamique et la validation côté client.
